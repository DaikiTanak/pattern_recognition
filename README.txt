# Pattern-Recognition

This zip file contains following files.
- EM.py This file applies EM algorithm on "x.csv". (GMM model) 
- VB.py This file applies Variational Bayes on "x.csv". (GMM model)

- z_EM.csv  Posterior probabilities of z of EM algorithm.
- z_VB.csv  Posterior probabilities of z of VB.

- params_EM.dat Parameters (pi, mu, sigma) of each clusters in each iteration.
- params_VB.dat Parameters (alpha, beta, m, W, nu) in each iteration.
